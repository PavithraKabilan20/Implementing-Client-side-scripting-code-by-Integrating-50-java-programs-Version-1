Calculating Simple Interest using Console

public class Main{
    public static void main(String[]args){
        
        System.out.print("Enter the value of Principle: ");
        String principle =System.console().readLine();
        
        System.out.print("Enter the value of number: ");
        String number=System.console().readLine();
        
        System.out.print("Enter the value of rate: ");
        String rate=System.console().readLine();
        
        float p=Integer.valueOf(principle);
        
        float n=Integer.valueOf(number);
        
        float r=Integer.valueOf(rate);
        
        float SI=p*n*r/100;
        
        System.out.println("The simple Interest for the given values are: "+SI);
      
    }
}


Input:

Enter the value of Principle: 1000
Enter the value of number: 4
Enter the value of rate: 10

Output:
The simple Interest for the given values are: 400
